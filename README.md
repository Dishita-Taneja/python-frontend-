🎂 Age Calculator (Streamlit)

This is a simple Age Calculator built using Streamlit. It allows users to enter their date of birth and calculates their exact age in years, months, and days.

✨ Features

	•	User-friendly date input
	•	Calculates:
	•	Age (Years, Months, Days)
	•	Time remaining until next birthday 🎉
	•	Handles edge cases like future dates
	•	Interactive UI using Streamlit

🧠 Logic (Next Birthday)

	•	Checks if the birthday has already passed this year
	•	If yes, shifts it to next year
	•	Calculates the remaining:
	•	Years
	•	Months
	•	Days
